# SCREENS
 

 
# Features

### 「旅にでよう。好きな居場所を見つけよう。」

好きな場所のシェアに特化したSNSアプリ
友達とつながったり、自分のいまいる場所を簡単にシェアすることができます。
フィーリングタグ機能で、気持ちをワンタップで表現でき、場所探しも簡単！
気になる場所を見つけたら「行きたい！」ボタンをタップしてチェックしておきましょう！
 
# Features

- Appleログイン
- Googleログイン
- 記事投稿機能
- カメラ撮影
- フォトライブラリからインポート
- 画像ダウンロード / アップロード
- タグ機能
- 行きたい場所のチェック
- 行った場所のチェック
